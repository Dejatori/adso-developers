DESARROLLO WEB -  ADSO SENA

Bienvenido a nuestro portal de práctica de desarrollo web, donde puedes explorar y experimentar con una variedad de tecnologías y herramientas. Desde HTML y CSS hasta PHP y JavaScript, aquí encontrarás recursos y ejercicios para mejorar tus habilidades de programación web.

Información

Esta plataforma de práctica de desarrollo web te servirá para familiarizarte con el desarrollo full stack de un sitio/aplicación web y poder implementarlo en tu proyecto.

¡Únete a este proyecto!

Todos los aprendices de análisis y desarrollo de software del Sena están invitados para contribuir al repositorio público de GitHub, donde juntos podemos aprender, colaborar y crear soluciones innovadoras.

Características

Estas características son fundamentales para proporcionar a nuestros usuarios una experiencia web excepcional y adaptada a sus necesidades.

Diseñada con Bootstrap 5

Nuestro marco de diseño es Bootstrap. Esto garantiza una interfaz de usuario moderna, limpia y altamente funcional. Con Bootstrap, puedes estar seguro de que tu experiencia de navegación será intuitiva y atractiva.

Personalizable a tu gusto

Valoramos la individualidad y la creatividad. Por eso, hemos diseñado nuestra página web para que puedas personalizarla según tus preferencias. Cambia los colores, las fuentes y otros elementos para que se adapten a tu estilo personal. ¡Haz que la página sea tuya!

Totalmente Responsiva

Es totalmente responsiva, lo que significa que se adapta perfectamente a dispositivos de todos los tamaños, ya sea que estés navegando en una computadora de escritorio, una tableta o un teléfono móvil. No importa dónde te encuentres, siempre tendrás una experiencia de usuario óptima.

Compatibilidad con todos los navegadores

Funciona sin problemas en todos los navegadores populares. Ya sea que uses Chrome, Firefox, Safari o Edge, nuestra página web funcionará a la perfección.

Herramientas que usamos para el desarrollo de esta web

En la creación de esta página web, hemos empleado una serie de herramientas y tecnologías poderosas para garantizar su calidad y funcionalidad. Estas son algunas de las herramientas esenciales que han sido parte integral de nuestro proceso de desarrollo.
